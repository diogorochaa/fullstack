# Fincheck Web (apps/web)

Frontend principal do Fincheck usando React + Vite + TypeScript.

## Scripts

```bash
# desenvolvimento
pnpm dev

# build de producao
pnpm build

# preview local do build
pnpm preview

# lint
pnpm lint

# typecheck
pnpm check-types
```

## Stack

- React 19
- React Router 7
- React Hook Form + Zod
- Tailwind CSS 4
- Vite 8
