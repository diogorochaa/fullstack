# @repo/eslint-config

Colecao de configuracoes ESLint compartilhadas no monorepo Fincheck.

## Conteudo

- `base.js`: regras base para projetos TypeScript
- `next.js`: regras para apps Next.js
- `react-internal.js`: regras para projetos React internos

Este pacote e consumido pelos apps em `apps/*` via workspace (`workspace:*`).
